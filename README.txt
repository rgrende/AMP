AMP: Athletics Music Player for Carroll College

Created by : Rakiah Grende, Robert Hereth, and Elaine Schultz
---------------------------------------------------------------------------

NOTE: This application is not complete and fully featured.



Before running the application:
- Install your preference of Java IDE (we recommend IntelliJ)
- Ensure your run configuration includes the files from the lib folder
- Run DBTools.java using this configuration to set up the database

To run the application:
- Run AMPGUI.java
- Please see the 'Documentation' tab under 'Help' in the menu bar for information on
    controls and functionality.
